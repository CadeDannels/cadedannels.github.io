This week we are going to work on embedding vega visualizations into our reveal.js slides. In the next few slides we will show 2 visualizations, interpret both of them, and show the code of how to reproduce the plots yourself.

Viz 1

When trying to interpret the visualization it is hard considering these are fake variables from the vega dateset. However, if you did try to analyze the plot you could say that D is the maximum value and H is second. The clear lowest value is G.

How to make the plot?


						< div id="vis">< /div>
			< script type="text/javascript">
				var spec = {
 
		  "$schema": "https://vega.github.io/schema/vega-lite/v5.json",
		  "description": "A simple bar chart with embedded data.",
		  "data": {
		    "values": [
		      {"a": "A", "b": 28}, {"a": "B", "b": 55}, {"a": "C", "b": 43},
		      {"a": "D", "b": 91}, {"a": "E", "b": 81}, {"a": "F", "b": 53},
		      {"a": "G", "b": 19}, {"a": "H", "b": 87}, {"a": "I", "b": 52}
		    ]
		  },
		  "mark": "bar",
		  "encoding": {
		    "x": {"field": "a", "type": "nominal", "axis": {"labelAngle": 0}},
		    "y": {"field": "b", "type": "quantitative"}
		  }

		};  /* JSON dump of your chart's spec */
		var opt = {"renderer": "canvas", "actions": false};  /* Options for the embedding */


		vegaEmbed("#vis", spec, opt);
